역할: 너는 전 세계 패키지 배포 및 프론트엔드 개발에 정통한 시니어 풀스택 개발자이자 UX/UI 디자이너야. 목표: 마케터를 위한 웹 애플리케이션인 '마케팅 인사이트 툴(Mkt Insight Tool)'을 HTML, CSS(Tailwind CSS), JavaScript(Vanilla JS) 단일 파일 또는 간단한 구조로 개발해줘. [앱 개요 및 콘셉트] - 이 앱은 마케터가 시장 데이터나 광고 성과, 경쟁사 동향 등을 입력하면 직관적인 대시보드와 인사이트를 제공하는 툴이다. - 복잡한 서버(Backend) 없이, 브라우저 로컬 환경(LocalStorage)이나 오픈 API를 활용하여 전면 프론트엔드 단에서 구동되도록 짜줘. - 디자인은 Modern, Clean, Professional 감성의 다크 모드/라이트 모드를 지원하는 대시보드 스타일이어야 해. (Tailwind CSS 활용) [핵심 기능 요구사항] 1. 대시보드 메인 화면: 주요 KPI 카드 3~4개 (예: CTR, ROAS, 전환수, 경쟁사 변동 지표 등 가상 데이터 반영), 트렌드를 볼 수 있는 깔끔한 차트 영역 (Chart.js 라이브러리 CDN 포함할 것). 2. 데이터 입력/업로드 섹션: 마케터가 직접 텍스트나 CSV 형식의 데이터를 붙여넣거나 입력할 수 있는 폼(Form). 3. 인사이트 자동 생성기: 입력된 데이터를 기반으로 조건문(또는 가상 로직)에 따라 "현재 광고 효율이 좋습니다", "경쟁사 배너 변동이 감지되었습니다" 같은 마케팅 액션 아이템을 텍스트로 깔끔하게 정리해 주는 화면. 4. 히스토리 저장: 입력하거나 분석한 기록이 브라우저를 새로고침해도 유지되도록 LocalStorage를 활용해줘. [기술 스택 및 제약사항] - 외부 프레임워크(React, Vue 등)를 쓰지 않고, 깃허브 페이지스(GitHub Pages)에 파일 하나만 올려도 바로 작동할 수 있게 'index.html' 단일 파일 구조(Single File Component 스타일)로 작성해줘. - CSS는 Tailwind CSS CDN을 ``에 포함해서 스타일링해줘. - 아이콘은 Lucide Icons 또는 FontAwesome CDN을 사용해줘. - 차트는 Chart.js CDN을 사용해줘. 코드를 작성할 때 생략(// 기존 코드 동일 등)하지 말고, 바로 복사해서 쓸 수 있는 완벽한 전체 HTML 코드를 출력해줘. 완성도 높은 UI/UX를 보여줘.
